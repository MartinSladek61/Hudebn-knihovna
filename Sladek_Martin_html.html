Úvod


Tato dokumentace obsahuje informace o mém projektu na předmět Značkovací jazyky.

Projekt popisuje aplikaci hudební knihovny s možností streamování hudby. V knihovně je 5 záznamů(písní) a každý z nich obsahuje elementy a atributy, jenž jej popisují a definují. V XSD souboru jsou pak popsány samotné elementy a atributy. Pomocí XSLT souboru se pak XML soubor transformuje do HTML.


Metodika

Jako vývojové prostředí jsem použil Microsoft Visual Studio, ve kterém jsem částečně použil generování XSD schématu ke XML souboru. Dále jsem použil Microsoft Visual Studio k vytvoření XSL souboru šablony pro transoformaci XML souboru. Ten jsem pak transformoval pomocí zdarma dostupného on-line transformátoru. Tato HTML dokumentace byla také vytvořena v Microsoft Visual Studiu.

Popis transformace

Transformuji nejzákladnější údaje o záznamech ze souboru XML tak, aby se daly poté zobrazit v HTML v tabulce. Používám k tomu účelu for-each cyklus, který mi pro každý výskyt elementu s názvem song vybere hodnoty požadovaných elementů a atributů.
